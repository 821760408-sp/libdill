NAME

ipaddr_str - convert address to a human-readable string

SYNOPSIS

**const char *ipaddr_str(const struct ipaddr *addr, char *ipstr);**

DESCRIPTION

Formats address as a human-readable string.

addr is the address to convert. ipstr is the buffer to store the result it. The buffer must be at least IPADDR_MAXSTRLEN bytes long.

RETURN VALUE

The function returns ipstr argument, i.e. pointer to the formatted string.

ERRORS

No errors.

EXAMPLE

char buf[IPADDR_MAXSTRLEN];
ipaddr_str(&addr, buf);
