NAME

ipaddr_setport - changes port number of the address

SYNOPSIS

void ipaddr_setport(struct ipaddr *addr, int port);

DESCRIPTION

Changes port number of the address.

RETURN VALUE

None.

ERRORS

No errors.

EXAMPLE

ipaddr_setport(&addr, 80);
